Size
===================

The size of the worksheet is defined by its width and height, which can be set in the properties of the worksheet. To simplify the handling of the different sizes that are commonly used, the following options and presets are available:

* **View Size**  - the size of the worksheet is set to the size of the current view and the whole available area on the screen provided for the worksheet view is used for the visualization, usefull when analyzing data without any need to work with fixed sizes for printing or exporting and where more space for the visualization is needed.
* **Standard Page** - the size of the worksheet is set to a standard page size, such as A4 or Letter, which is useful for printing or exporting the visualization in a standard format.
* **Custom Size** - the size of the worksheet can be set to any custom dimensions, which is useful for specific use cases where a non-standard size is required.

The default size of the worksheet is set to "Custom Size" with the size 10cm x 10cm. The default value can be changed and the changes can be persisted as new default values as described in the section :ref:`themes_templates`
