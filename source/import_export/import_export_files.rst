Import from Files
===================

LabPlot supports the import of data from various file formats. The current list of supported formats:

- ASCII
- Binary
- SAS
- Stata
- SPSS
- MATLAB
- OpenDocument Spreadsheets (ods)
- Excel (xlsx)
- HDF5
- FITS
- netCDF
- ROOT (CERN)
- LTspice
- Ngspice
- Binary Logging Format (BLF)
